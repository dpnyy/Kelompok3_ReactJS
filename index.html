Kelebihan React JS

1.Lebih mudah dipahami

Penulisan coding yang deklaratif, sehingga kita dapat memahami kode javascript dengan mudah. Terutama ketika memeriksa kesalahan pada penulisan kode, sehingga proses debugging menjadi lebih mudah. Dengan framework yang sama, kita juga dapat membuat tampilan web atau aplikasi yang menarik, tetapi tetap sederhana.

2.Pemakaian berulang

Kita bisa mengambil semua bagian coding javascript atau sebagian saja, lalu mengembangkannya sesuai kebutuhan masing-masing. Hal ini akan menghemat banyak waktu dalam membuat UI yang diinginkan, sehingga pekerjaan menjadi lebih cepat selesai

3.Cakupan untuk Pengujian Kode

Aplikasi ReactJS sangat mudah untuk diuji. Ini menawarkan ruang lingkup di mana pengembang dapat menguji dan men-debug kode mereka dengan bantuan alat asli.

4.SEO Friendly

Kerangka JavaScript tradisional memiliki masalah dalam menangani SEO. Mesin pencari umumnya mengalami kesulitan dalam membaca aplikasi yang berat JavaScript. Banyak web developer yang sering mengeluhkan masalah ini. ReactJS mengatasi masalah ini yang membantu pengembang untuk dinavigasi dengan mudah di berbagai mesin pencari. Itu karena aplikasi React.js dapat berjalan di server, dan DOM virtual akan merender dan kembali ke browser sebagai halaman web biasa.

Kelemahan React JS

1. Dokumentasi yang Buruk

Teknologi React memperbarui dan mempercepat dengan sangat cepat sehingga tidak ada waktu untuk membuat dokumentasi yang tepat. Untuk mengatasinya, pengembang menulis instruksi mereka sendiri dengan mengembangkan rilis baru dan alat dalam proyek mereka saat ini.

2. View Layer

React JS merupakan library javascript dalam bentuk view layer. Walaupun penggunaannya sangat mudah, tetapi framework ini tidak cocok untuk membangun aplikasi yang besar. Sehingga kita tetap harus melengkapinya dengan router, data layer, struktur dan lainnya secara terpisah. Anda bisa memakai library luar untuk melengkapi hal tersebut. Namun, hal ini tidak praktis dan akan menyulitkan para pengguna baru.

3. Kompatibilitas

Library javascript ini dapat digunakan untuk browser versi terbaru dan tidak bisa pada versi lama. Sehingga mengharuskan kita untuk meng-update browser dahulu agar dapat memakai react JS.
