React JS

React Js adalah sebuah pustaka/library javascript yang bersifat open source untuk membangun User Interface. React Js memiliki sifat composable user yang berarti pengguna bisa membuat berbagai UI, oleh karena itu React Js sangat populer dikalangan Web Development. React JS dibuat oleh Jordan Walke dan telah resmi dirilis pada tahun 2013, yang dimana Jordan Walke merupakan salah satu karyawan di perusahaan Facebook.

React Js banyak dibicarakan pada saat ini, bahkan dikatakan sebagai framework paling populer di masa sekarang. Namun nyatanya React JS bukalah sebuah framework, melainkan sebuah pustaka yang digunakan untuk membangun user interface. UI yang yang mampu dibuat oleh React JS berbentuk interaktif, sehingga model desain yang digunakan biasanya simple untuk setiap state dalam aplikasinya. Declarative View yang ia miliki mampu membuat kode lebih mudah untuk diprediksi dan mudah di debug.

Kegunaan React JS

React Js digunakan dalam membangun User Interface pada bagian tampilan dan logika disekitarnya. Tampilan desain sederhana yang dapat digunakan dalam pengembangan aplikasi berbasis web, React JS juga memiliki fitur unggulan sebagai berikut :

1. Declarative

React dapat digunakan untuk membuat desain simple untuk setiap state pada aplikasi sehingga UI lebih interaktif, Declarative views juga membuat code mudah untuk di debug.

2. Component Based

Dapat membuat Encapsulated Component yang dapat mengatur setiap tahapannya sehingga dapat membuat complex UI

3. Learn Once, Write anywhere

Developer dapat mengembangkan fitur terbaru tanpa mengubah code yang ada, pengerjaan dapat menggunakan node JS dan React Native untuk mobile apps

Performansi React JS

React Js dikenal sebagai salah satu pengembang web tercepat dimana react secara aktif menggunakan pengembang single page application, juga dianggap sebagai pilihan optimal untuk desain interaktif, serta salah satu contoh terbaik di kinerja renderingnya


Penggunaan Reactjs meminimalisir jumlah operasi DOM yang boros sehingga menghasilkan antarmuka pengguna yang cepat, Reactjs menggunakan virtual DOM dalam penggunannya dimana pada DOM pengubahan sedikit element membutuhkan rendering seluruh page namun tidak pada virtual DOM

Kelebihan React JS

1.Lebih mudah dipahami

Penulisan coding yang deklaratif, sehingga kita dapat memahami kode javascript dengan mudah. Terutama ketika memeriksa kesalahan pada penulisan kode, sehingga proses debugging menjadi lebih mudah. Dengan framework yang sama, kita juga dapat membuat tampilan web atau aplikasi yang menarik, tetapi tetap sederhana.

2.Pemakaian berulang

Kita bisa mengambil semua bagian coding javascript atau sebagian saja, lalu mengembangkannya sesuai kebutuhan masing-masing. Hal ini akan menghemat banyak waktu dalam membuat UI yang diinginkan, sehingga pekerjaan menjadi lebih cepat selesai

3.Cakupan untuk Pengujian Kode

Aplikasi ReactJS sangat mudah untuk diuji. Ini menawarkan ruang lingkup di mana pengembang dapat menguji dan men-debug kode mereka dengan bantuan alat asli.

4.SEO Friendly

Kerangka JavaScript tradisional memiliki masalah dalam menangani SEO. Mesin pencari umumnya mengalami kesulitan dalam membaca aplikasi yang berat JavaScript. Banyak web developer yang sering mengeluhkan masalah ini. ReactJS mengatasi masalah ini yang membantu pengembang untuk dinavigasi dengan mudah di berbagai mesin pencari. Itu karena aplikasi React.js dapat berjalan di server, dan DOM virtual akan merender dan kembali ke browser sebagai halaman web biasa.

Kelemahan React JS

1. Dokumentasi yang Buruk

Teknologi React memperbarui dan mempercepat dengan sangat cepat sehingga tidak ada waktu untuk membuat dokumentasi yang tepat. Untuk mengatasinya, pengembang menulis instruksi mereka sendiri dengan mengembangkan rilis baru dan alat dalam proyek mereka saat ini.

2. View Layer

React JS merupakan library javascript dalam bentuk view layer. Walaupun penggunaannya sangat mudah, tetapi framework ini tidak cocok untuk membangun aplikasi yang besar. Sehingga kita tetap harus melengkapinya dengan router, data layer, struktur dan lainnya secara terpisah. Anda bisa memakai library luar untuk melengkapi hal tersebut. Namun, hal ini tidak praktis dan akan menyulitkan para pengguna baru.

3. Kompatibilitas

Library javascript ini dapat digunakan untuk browser versi terbaru dan tidak bisa pada versi lama. Sehingga mengharuskan kita untuk meng-update browser dahulu agar dapat memakai react JS.

Dasar-dasar React JS

1. JSX

JavaScript XML (JSX) merupakan ekstensi sintaks khusus dan valid dalam ReactJS. yang memungkinkan pembuatan file HTML tidak terpisah. HTML dan JavaScript dapat digabung dan dikerjakan pada file yang sama.

2.Component

Komponen adalah blok kode independen yang dapat digunakan kembali yang membagi UI menjadi bagian-bagian yang lebih kecil. React memiliki dua jenis komponen: functional dan class.

Functional Component :

  • merupakan sebuah JavaScript/ES6 function
  • return a React element (JSX)
  • dimulai dengan huruf kapital
  • menggunakan prop sebagai parameter

    • Class Component :

  • merupakan ES6 class, menjadi sebuah komponen jika dilakukan 'extend' React Component
  • menggunakan Props (pada constructor) jika dibutuhkan
  • harus memiliki metode render() ketika return JSX

    • 3. Props

    React memiliki objek khusus yang disebut prop (singkatan dari property) yang digunakan untuk mengangkut data dari satu komponen ke komponen lainnya. Props merupakan nilai khusus yang dapat menjadikan komponen lebih dinamis.

    4. State.

    State.js adalah pustaka JavaScript untuk menyematkan status kelas satu yang dapat dikomposisi dan dapat diwariskan ke dalam arbitrary owner objects.
